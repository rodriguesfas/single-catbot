# NLP

## Install manual

    pip install fastapi uvicorn chatterbot chatterbot-corpus

## Install auto

    pip install -r requirements.txt


## Run

    uvicorn main:app --reload

or 

    uvicorn main:app --reload --host="0.0.0.0" --port="8000"
